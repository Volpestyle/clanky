# CLANKY_* environment variable reference

Generated by `pnpm env:reference` (scripts/generate-env-reference.ts) from
the CLANKY_* reads in agent/, bin/, scripts/, and packages/. Do not edit by
hand — edit the description table in the generator and regenerate.

Values are read from the process environment first, then `.env.local` at
the repo root (process.env wins; see agent/lib/env-store.ts).

## Core paths & lifecycle

| Variable | Description |
| --- | --- |
| `CLANKY_BRAIN_AGENT` | Herdr agent name for the command-host/brain pane (default `clanky`). |
| `CLANKY_HERDR_BIN` | herdr binary the lifecycle script uses (default `herdr` on PATH). |
| `CLANKY_HOME` | Clanky data root for durable state (default `~/.clanky`): auth store, transcripts, memory, push registry, media. |
| `CLANKY_MAIN_AGENT` | Herdr agent name of the main face pane, for gateway bridge commands (default `clanky`). |
| `CLANKY_REPO_DIR` | Repo checkout dir; anchors `.env.local`, the spawn seam's `bin/clanky.ts`, and mirror scripts when a process is not started from the repo root. |
| `CLANKY_SESSION` | Herdr session name the lifecycle script targets (default `clankies`). |
| `CLANKY_UP_SESSION_TIMEOUT_MS` | How long `clanky up` waits for the herdr session to start (default 15000). |

## Eve server address & supervision

| Variable | Description |
| --- | --- |
| `CLANKY_EVE_BASE_URL` | Base URL other processes use to reach the eve server (default `http://127.0.0.1:<port>`); always a URL. |
| `CLANKY_EVE_BIND_HOST` | Interface the owned eve dev server binds to (e.g. `127.0.0.1`, `0.0.0.0`); never a URL. See agent/lib/eve-address.ts. |
| `CLANKY_EVE_CALLBACK_PROXY` | Set `0` to disable the localhost OAuth callback proxy the face runs for connection auth flows. |
| `CLANKY_EVE_CALLBACK_PROXY_PORT` | Port for the OAuth callback proxy (default 3000). |
| `CLANKY_EVE_HEALTH_POLL_MS` | Face brain-health poll interval (default 5000). |
| `CLANKY_EVE_HEALTH_TIMEOUT_MS` | How long the face/CLI/lifecycle wait for a starting brain to become healthy (default 180000). |
| `CLANKY_EVE_HOST` | Legacy combined var: an http(s) URL is treated as CLANKY_EVE_BASE_URL, a bare host as CLANKY_EVE_BIND_HOST. Prefer the split vars. |
| `CLANKY_EVE_KILL_TIMEOUT_MS` | Wait after SIGKILL when stopping the owned eve server (default 2000). |
| `CLANKY_EVE_PORT` | eve/relay port (default 2000). |
| `CLANKY_EVE_PROBE_TIMEOUT_MS` | Per-probe fetch timeout for /eve/v1/info health checks (default 2000). |
| `CLANKY_EVE_RESTART_STOP_TIMEOUT_MS` | Shorter SIGTERM grace used for intentional config-change restarts (default 1000). |
| `CLANKY_EVE_STOP_TIMEOUT_MS` | SIGTERM grace before SIGKILL when stopping the owned eve server (default 5000). |
| `CLANKY_RESTART_ATTACHED_EVE` | Set `1` to let the face restart an attached (non-owned) eve dev server after config writes, even when it is not an ancestor process. |
